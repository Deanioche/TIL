## **1. 어휘 분석**

- 프로그램 분석
    ```c
    int x12;
    x12 = 1 + 5 * 2;
    if x12>10 then ...
    ```
    위와 같이 가상의 프로그램이 있다면 컴퓨터는 이 프로그램을 보고 실행하기 위해 분석한다.

    1. 문자의 분석
        - i, n, t, x, 1, 2, ;, =, +, 5, *, f, >, …

    2. 어휘의 분석
        - int, x12, ;, =, 1, +, 5, if, >, 10, then, …

    3. 구문의 분석
        - <선언문> ::= <자료형> <변수> ;
        - <대입문> ::= <변수> = <수식> ;

- 어휘 분석

    - 프로그램에서 사용된 단어를 구별해 냄
    - 토큰:어휘 분석을 통해 얻어지는 결과
        - 연산자,구분자,식별자,예약어 등
            - 연산자 : +, -, *, /, =, ...
            - 구분자 : ,(콤마), ;, [, ], ...
